【PUREREF】-快捷键基本操作

一、软件基本操作

官方使用手册：https://www.pureref.com/handbook/

下载地址：https://www.pureref.com/download.php

这里是内容。

1.软件窗口

移动软件窗口：鼠标右键拖动

软件窗口自定义大小（高宽）：鼠标悬停软件窗口边缘，出现图标后右键拖动自定义大小（高宽）

将窗口重置为选中图片大小： Ctrl+Shift+R（软件窗口适应选中图像的大小, 图像可多选）

移动软件界面：软件框内右键拖拽软件，可以把它移到屏幕上合适的位置。

2.视图

保持软件置于（其他软件）顶层：Ctrl+Shift+A（重复操作取消）

保持软件置于（其他软件）底层：Ctrl+Shift+B（重复操作取消）

全屏显示/最大化： Ctrl+F （重复操作取消/鼠标右键拖动）

最小化： Ctrl+M （任务栏点击软件图标返回）

窗口锁定： Ctrl+W（重复操作取消，仅固定软件窗口，不能移动/调节窗口大小，其他调节/导入图片或移动画布等不受影响）

新的窗口：Ctrl+K （新的窗口会清除当前画布内容，软件会提示你保存。）如果想同时存在两个参考图窗口，建议直接开一个新的软件，不要通过这个快捷键，就可以同时存在两个参考图窗口。

居中放大一个或多个图片：双击左键/空格（重复操作返回）

小键盘→/←：看上一个/下一个图片

特殊的视图方式：

(1)对鼠标透明：Ctrl+T

相当于鼠标可以穿透软件，不会发生选中的状态，包括对已放入的参考图。可以和全屏显示叠加。

关闭的方法：

① 鼠标悬停在任务栏的软件图标，并再次使用快捷键Ctrl+T可解除状态 。
② 鼠标悬停在任务栏的软件图标, 小窗会出现设置小齿轮在内的几个图标，选 Disable Transparent To Mouse Mode 图标可解除状态。
（2）背景透明： Ctrl+Y

选中任意图片，重复操作恢复。可隐藏软件背景，只显示图片。和【对鼠标透明】的区别在于，不会选中软件背景，但是可以任意对图片进行操作，很方便。

3.画布

放大画布：鼠标滚轮 / 鼠标左键+Z

移动画布：鼠标中键 / 鼠标左键+Alt

锁住画布：Ctrl+R（重复操作取消）锁住画布，不能编辑、导入图片，可以移动、缩放软件窗口。

输出画布： Ctrl+E （以图片形式输出画布）

输出选择图像：Ctrl+Alt+I （以图片形式输出选中图像）

优化画布： Ctrl+O

优化画布排列： Ctrl+Shift+P（重新排列并优化画布）

4.透明度/背景

这些操作只是改变了背景的颜色/透明度，鼠标还是会点选到软件背景。和上面介绍的两种特殊视图不一样。

增加/降低软件不透明度： Ctrl + ±（加减号）

自带预设背景（可以去设置自定义）：

黑暗背景：ALT+1

白色背景：ALT+3

透明背景：ALT+2

5.加入文本注释

输入文本：Ctrl+N

①粘贴或创建新文本的同时，选择一个图像文本会作为图像的子级存在。

②在画布空白区域粘贴或创建新文本则文本独立存在。

更改文本及文本背景颜色

编辑注释时，单击顶部的 T 按钮可更改文本颜色和不透明度。

编辑笔记时，单击底部的 T 按钮可更改笔记的背景颜色和不透明度。

6.其他常用快捷键

多选：按住shift+左键选择多张/左键框选

全选：Ctrl+A

复制：Ctrl+C

粘贴：Ctrl+V

删除：Delete

退回上一步：Ctrl+Z

撤回并重做上一步：Ctrl+Shift+Z

保存：Ctrl+S

另存为：Ctrl+Shift+S

关闭软件：Ctrl+X

打开图片所在目录：Ctrl+Shift+O（选择图片在操作）从其他软件复制粘贴过来的图片无效。

加载图像： Ctrl+I

显示图像坐标： D+左键

取色器：S+左键(对图像，会显示色值、RGB等)

设置：Ctrl+U

二、编辑图像

1.添加图像的方法

直接拖图片进去（可以直接网站拖图很方便） / Ctrl+I导入图片

从外部Ctrl+C复制图片，然后Ctrl+v粘贴图片。

重新链接丢失的图像：Ctrl+Alt+Shift+R

如果禁用嵌入本地图像并且在磁盘上找不到图像，将显示问号而不是图像。如果图像被移动或重命名，则可以重新链接。触发此操作或双击图像以选择新来源。只能链接磁盘上的本地图像。从剪贴板粘贴或从网站中添加的图像将被嵌入。

2.选择一个或多个图片

(1)单选：单击左键。

(2)多选：左键框选。

(3)多选：按住shift+左键选择多张

(4)选择全部：Ctrl+A

3.缩放（可多选图片进行）

缩放：鼠标中键

平滑缩放图片：Ctrl+Alt+左键

4.裁剪编辑

常用：

裁剪图片：按住C键不放+左键，框选图片需要的区域。

取消剪裁/复位图像：选中图片，Ctrl+Shift+C，或直接Ctrl+Z回退上一步

进阶：

在图像自身框内移动图像：V+左键

在图像自身框内放大图像：Shift+V+左键

5.旋转/翻转

旋转图片：Ctrl+左键旋转图片，+shift 会45°旋转。

简单翻转图片：Shift+Alt+鼠标左键（上下滑动=上下翻转，左右滑动=左右翻转）

上下翻转：Shift+Alt+V

左右翻转：Shift+Alt+H

复位变换：Ctrl+Shift+T

6.移动

移动图片：左键拖拽图片

限制图像以直线（上下左右）移动：（选中图像后）Shift+左键

限制图像移动到其他图像边缘：（选中图像后）Shift+空格+左键

7.排列

按图片最佳缩放大小排列：Ctrl+P

有序的排列：Ctrl+Shift+p

按时间排列：Ctrl+Alt+A

按名字排列：Ctrl+Alt+N

堆叠图片：Ctrl+Alt+S（把所有图片堆叠一起）

8.图层/对齐

图片图层上移/下移：（选中图片）方向键↑/↓箭头

图片上下左右对齐：（选中图片）Ctrl+方向键↑/↓/←/→

多张图片同宽/高：（选中图片）Ctrl+Alt+方向键↑/↓

图片上下居中对齐：（选中图片）Ctrl+Alt+Shift+方向键↑/↓

9.透明度/颜色/灰度

所有图片灰度转换： Ctrl+Alt+G（重复操作恢复）

选择图片灰度转换： ALT+G（可多选）

选择图像变化不透明度： Ctrl+Alt+Shift+左键（重复操作恢复）

10.相机

复位摄像机： Ctrl+G

整体平滑变焦 Z+左键

选择图片平滑变焦：Shift+V+左键 (让图像在其本身的图像框内放大，相当于放大图片但是不影响其他的图片)

三、自定义软件

设置：Ctrl+U

页面自定义部分：

Background Color 背景颜色——画布周围的区域。

Canvas Color 画布颜色- 图像的背景。

Border Color 边框颜色——主窗口的边框颜色。

Text Color 文字颜色- 主要文字颜色。

Text Hover Color 文本悬停颜色- 悬停在可点击文本上的颜色。

Secondary Text Color - 二级文字颜色

Menu Background Color 菜单背景颜色——想要菜单中的颜色与画布中的颜色不同吗？放在这里。

Menu Border Color 菜单边框颜色- 菜单边框和按钮颜色。想要在菜单边框上使用与主窗口不同的颜色吗？放在这里。

Selection Color 选择颜色- 选择矩形和所选图像边框的颜色。

Master Opacity 主界面不透明度

设置完记得【Save to Preset】否则关闭软件就没了，一共可以保存四个自定义栏位。
